# 🐴 Pony express 

Pony_Express est un utilitaire qui exporte vos donnees GRIST, les mets en page et genere un PDF pour chaque ligne de votre tableau

It was first created to handle Erasmus paperwork but was designed to be reusable.

See our [documentation](https://erasmus-plus-7899ee.gitlab.io/tutorial/1_first_run/) to configure your own GRIST to PDF tool !
